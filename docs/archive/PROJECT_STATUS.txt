ARCHITECTURE SUMMARY
================================================================================

Frontend (Next.js):
  - Dashboard page at /dashboard/ia
  - 4 tabs: Models, Training, Journalists, Publication Style
  - AIFieldHelper component (universal ✨ button)
  - Sparkles icon on every CMS field

API Layer (Next.js Routes):
  - POST /api/ai/train - Analyzes published articles
  - GET/POST /api/ai/settings - Configuration
  - POST /api/ai/generate - Claude content generation
  - All multi-tenant isolated via Supabase

Data Layer (Supabase PostgreSQL):
  - ai_settings - Model configuration per tenant
  - journalist_profiles - Per-journalist analysis (style, tone, keywords)
  - publication_styles - Publication-wide style analysis
  - RLS policies for security

External:
  - Anthropic Claude 3.5 Sonnet API (content generation)
  - Supabase (database, auth, RLS)
